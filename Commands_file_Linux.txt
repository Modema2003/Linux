Команды для работы в Linux терминале

Файлы

Ls [ключ][файл]lsСписок файлов в каталогеls -lСписок файлов в каталоге в табличном видеls –a Список файлов в каталоге, включая скрытые папкиls   -authorСписок файлов + имя создателяls -clСписок файловс сортировкой по имениls -ctСписок файловс сортировкой по времени создания (сначала самые новые)ls -sСписок файловс размером файлаls -SСписок файловс сортировкой по размерам (сначала большие)ls *Список текстовых файловls –l >file.txtЗапись списка файлов в file.txtpwdПоказать, где находишьсяcd file.txtПеремещение в файл file.txtcd ..Перемещение на папку вверхcd -Перемещение в предыдущую папку, в которой были (вверх или вниз)cd /Перемещение в корневую папкуcat [ключ][файл]cat fileФайлfileвыводится в терминалcat file1 file2 > file3
Сцепление (слияние) файлов file1 и file2, после чего они помещаются в файл file3cat file1 >> file2
Присоединение файла file1 к концу файла file2cat -b file1
Вывод содержимого файла с нумерациейстрок, но без нумерации пустых строкcat -n file1
Нумерация всех строкmkdirпапкаСоздать каталог (папку)mkdirпапка1 папка2 Создать 2 папкиmkdir{папка1,папка2}
Создать 2 папки (запись без пробелов)rmdirпапка1Удалить папку1rm –d папка1Удалить папку1Touchfile1Создать файл1Touchfile1 file2Создать 2 файлы>file1Создать файл1>папка/файлСоздать файл в папке (путь)rmfile1Удалить файл1copy file1 file2Копировать file1 с новым именем file2morefile1выводит текста из файла на экран по 24 строки, нажатие  [Пробел]  выводит очереднуюпорции. Досрочно завершает работу при нажатии на клавишу [Q]lessfile1выводит текста из файла на экран, поддерживает возможности прокрутки вверх и поиска. Завершает работу при нажатии на клавишу [Q]find file1Поиск в данном каталогеfindfile1 (что) папка(где)Поиск по заданному путиfind / -name f|ile1Поиск по всем каталогам (с sudo) mvfile1 file2Переименовать file1 в file2sudo !!Повтор предыдущей команды
